## Architecture Overview

FuelPay follows **Clean Architecture** with three distinct layers:

### 1. Domain Layer (Pure Business Logic)
```
domain/
├── entities/      # Immutable business objects
├── repositories/  # Abstract interfaces
└── usecases/      # Single business operations
```

### 2. Data Layer (Implementation)
```
data/
├── datasources/   # Remote (Dio) & Local (Hive)
├── models/        # JSON-serializable (Freezed)
├── repositories/  # Implement domain interfaces
└── mappers/       # Model ↔ Entity conversion
```

### 3. Presentation Layer (UI)
```
presentation/
├── pages/         # Full-screen widgets
├── widgets/       # Reusable components
├── providers/     # Riverpod providers
└── notifiers/     # StateNotifier logic
```

## Code Standards

### Naming Conventions
- **Files**: snake_case (`user_repository.dart`)
- **Classes**: PascalCase (`UserRepository`)
- **Variables/Methods**: camelCase (`userName`, `fetchUser()`)
- **Constants**: UPPER_SNAKE_CASE or camelCase

### Architecture Rules

✅ **DO:**
- Keep domain layer pure (no external dependencies)
- Use immutable models (@freezed)
- Implement repositories as interfaces in domain
- Use Riverpod for state management
- Handle errors with custom exceptions
- Write unit/widget tests
- Use const constructors
- Follow single responsibility principle

❌ **DON'T:**
- Import presentation/data in domain
- Use mutable state
- Ignore error handling
- Create God classes
- Skip tests
- Mix business logic in UI

### File Organization Template

```dart
class YourRepository {
```

## Common Patterns

### Creating a Provider
```dart
// Data provider
final userRepositoryProvider = Provider((ref) {
  final dio = ref.watch(dioProvider);
  return UserRepositoryImpl(dio);
});

// Usecase provider
final getUserUsecaseProvider = Provider((ref) {
  final repo = ref.watch(userRepositoryProvider);
  return GetUserUsecase(repo);
});

// State notifier provider
final userNotifierProvider = 
  StateNotifierProvider<UserNotifier, AsyncValue<UserEntity>>((ref) {
    final usecase = ref.watch(getUserUsecaseProvider);
    return UserNotifier(usecase);
  });
```

### Handling Async Data
```dart
final userState = ref.watch(userNotifierProvider);

userState.when(
  data: (user) => UserWidget(user: user),
  loading: () => LoadingWidget(),
  error: (error, stack) => ErrorWidget(
    message: error.toString(),
    onRetry: () => ref.refresh(userNotifierProvider),
  ),
);
```

### Error Handling
```dart
class UserRepositoryImpl implements UserRepository {
  @override
  Future<UserEntity> getUser(String id) async {
    try {
      final model = await _datasource.getUser(id);
      return UserMapper.toEntity(model);
    } on DioException catch (e) {
      if (e.response?.statusCode == 404) {
        throw NotFoundException('User not found');
      }
      throw NetworkException('Failed to fetch user');
    } catch (e) {
      throw UnknownException('Unknown error');
    }
  }
}
```
